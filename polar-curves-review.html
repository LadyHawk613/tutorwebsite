Polar coordinate system

What are polar coordinates?

A point is described as \((r,\,\theta)\) where \(r\) is the distance from the pole (origin) and \(\theta\) is the angle measured counterclockwise from the polar axis (positive \(x\)-axis).

Converting between systems
Polar → Rectangular
$$x = r\cos\theta$$
$$y = r\sin\theta$$
Rectangular → Polar
$$r = \sqrt{x^2 + y^2}$$
$$\theta = \arctan\!\left(\frac{y}{x}\right)$$
⚠ When using \(\theta = \arctan(y/x)\), always check the quadrant. Add \(\pi\) if the point is in Q2 or Q3.
Negative \(r\) values

If \(r < 0\), the point is plotted in the direction opposite to \(\theta\).

$$(r,\,\theta) \;=\; (-r,\,\theta+\pi)$$

Example: \(\bigl(-3,\,\tfrac{\pi}{4}\bigr)\) is the same point as \(\bigl(3,\,\tfrac{5\pi}{4}\bigr)\).

Multiple representations

Every point has infinitely many polar representations.

$$(r,\,\theta) \;=\; (r,\,\theta + 2\pi k)\quad\text{for any integer }k$$
$$(r,\,\theta) \;=\; (-r,\,\theta + \pi + 2\pi k)$$
Symmetry tests
Polar axis
Replace \(\theta\) with \(-\theta\). If the equation is unchanged, the graph is symmetric about the polar axis (\(x\)-axis).
\(\theta = \dfrac{\pi}{2}\)
Replace \(\theta\) with \(\pi - \theta\). If unchanged, the graph is symmetric about the line \(\theta = \tfrac{\pi}{2}\) (\(y\)-axis).
Pole
Replace \(r\) with \(-r\). If unchanged, the graph is symmetric about the pole (origin).

These tests are sufficient but not necessary — failing a test does not guarantee the graph lacks that symmetry.

Curve types you need to know

Circles
Centered at pole
$$r = a$$

Circle of radius \(|a|\) centered at the origin.

Passing through pole
$$r = 2a\cos\theta \quad\text{or}\quad r = 2a\sin\theta$$

Diameter \(= 2|a|\). Center on \(x\)-axis (cos) or \(y\)-axis (sin).

Limaçons
$$r = a \pm b\cos\theta \qquad\text{or}\qquad r = a \pm b\sin\theta$$

Shape depends on the ratio \(a/b\) (assume \(a,b>0\)):

\(\dfrac{a}{b} < 1\)
Inner loop — has a loop inside. The curve passes through the pole twice.
\(\dfrac{a}{b} = 1\)
Cardioid — heart-shaped, passes through the pole exactly once.
\(1 < \dfrac{a}{b} < 2\)
Dimpled limaçon — has an indentation but no inner loop.
\(\dfrac{a}{b} \ge 2\)
Convex limaçon — smooth oval shape, no dimple or loop.

\(\cos\theta\) version → oriented along \(x\)-axis  |  \(\sin\theta\) version → oriented along \(y\)-axis

Rose curves
$$r = a\cos(n\theta) \qquad\text{or}\qquad r = a\sin(n\theta)$$
\(n\) odd
\(n\) petals, each of length \(|a|\). One petal lies along the axis.
\(n\) even
\(2n\) petals, each of length \(|a|\). Petals appear between and on the axes.

Petal tip length is always \(|a|\) regardless of \(n\). Period is \(\pi\) when \(n\) is odd; \(2\pi\) when \(n\) is even.

$$r = 3\cos(2\theta) \;\Rightarrow\; 4\text{ petals, each length }3$$
$$r = 2\sin(3\theta) \;\Rightarrow\; 3\text{ petals, each length }2$$
Lemniscates
$$r^2 = a^2\cos(2\theta) \qquad\text{or}\qquad r^2 = a^2\sin(2\theta)$$

Figure-eight shaped curve. Only exists where the right side \(\ge 0\).

$$r^2 = a^2\cos(2\theta)$$

Lobes along the \(x\)-axis (polar axis)

$$r^2 = a^2\sin(2\theta)$$

Lobes along \(\theta = \tfrac{\pi}{4}\) and \(\theta = \tfrac{5\pi}{4}\)

Archimedean spiral
$$r = a\theta$$

As \(\theta\) increases, \(r\) increases proportionally. Each full revolution adds \(2\pi a\) to the radius.

Lines & special curves
Horizontal
\(r\sin\theta = k \;\Rightarrow\; y = k\)
Vertical
\(r\cos\theta = k \;\Rightarrow\; x = k\)
Through pole
\(\theta = c\) — straight line at constant angle through the origin

Formulas & reference

Key angle values to memorize
\(\theta\) \(0\) \(\dfrac{\pi}{6}\) \(\dfrac{\pi}{4}\) \(\dfrac{\pi}{3}\) \(\dfrac{\pi}{2}\) \(\pi\) \(\dfrac{3\pi}{2}\)
\(\cos\theta\) \(1\) \(\dfrac{\sqrt{3}}{2}\) \(\dfrac{\sqrt{2}}{2}\) \(\dfrac{1}{2}\) \(0\) \(-1\) \(0\)
\(\sin\theta\) \(0\) \(\dfrac{1}{2}\) \(\dfrac{\sqrt{2}}{2}\) \(\dfrac{\sqrt{3}}{2}\) \(1\) \(0\) \(-1\)
Finding intercepts & zeros
Zeros (pole)
Set \(r = 0\) and solve for \(\theta\). These are angles where the curve passes through the origin.
Axis crossings
Evaluate \(r\) at \(\theta = 0,\;\tfrac{\pi}{2},\;\pi,\;\tfrac{3\pi}{2}\). The \(r\) values give distances on each axis.
Maximum \(|r|\)
Find where \(|\cos(n\theta)| = 1\) or \(|\sin(n\theta)| = 1\). Maximum petal length \(= |a|\).
Rate of change in polar

Treat \(r\) as a function of \(\theta\) and compute \(\dfrac{dr}{d\theta}\) to analyze how the distance from the pole changes.

$$\frac{dr}{d\theta} > 0 \;\Rightarrow\; r\text{ increasing (moving away from pole)}$$
$$\frac{dr}{d\theta} < 0 \;\Rightarrow\; r\text{ decreasing (moving toward pole)}$$

Example: For \(r = 2 + 3\cos\theta\), we have \(\dfrac{dr}{d\theta} = -3\sin\theta\). So \(r\) is increasing when \(\sin\theta < 0\), i.e., \(\theta \in (\pi,\,2\pi)\).

Graphing strategy (step by step)
Step 1
Identify curve type from the equation form (cardioid, rose, limaçon, etc.).
Step 2
Find key values: zeros (\(r=0\)), maximum \(r\), and axis intercepts.
Step 3
Apply symmetry tests to reduce the number of points needed.
Step 4
Build a table of \((\theta,\, r)\) values for \(\theta \in [0,\,2\pi]\), then plot on polar axes.
Step 5
Verify petal count and orientation match expected values for the curve type.
